# lesson_git_10
Тренировка с командами git

## Описание
Этот репозиторий создан для практики базовых операций Git:
- Создание локального репозитория
- Связывание с удаленным репозиторием на GitHub
- Работа с ветками
- Слияние изменений
- Отправка изменений на GitHub

## Файлы
- `README.md` - описание проекта
- `git_commands.md` - справочник по основным командам Git
